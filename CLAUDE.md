# CLAUDE.md — Инструкции для AI

## Общая цель проекта
**Wildberries Product Analysis** — инструмент для селлеров Wildberries.  
Собирает данные из нескольких источников (WB кабинет, MPStats, Google Sheets — лист **Unit**), объединяет их и проводит глубокий **ИИ-анализ** товаров и рекламных кампаний.

## Ключевые возможности
- Глубокий анализ товара по артикулу
- Сбор и анализ статистики (продажи, выручка, позиция, реклама и т.д.)
- Планируется: мощный анализ рекламных кампаний
- Экспорт / работа с Google Sheets
- ИИ-анализ (Groq) на основе собранных данных

## Целевая аудитория
Селлеры Wildberries (продавцы на маркетплейсе)

## Текущий статус
- Хорошо работает вытягивание данных из источников
- Основной фокус сейчас — улучшение анализа и добавление новых типов анализа

## Технологии
- Next.js 15 (App Router) + TypeScript
- Tailwind + shadcn/ui
- Groq (основной LLM для анализа)
- MPStats API
- Wildberries API
- Google Sheets API

## Критически важные правила
- **ЗАПРЕЩЕНО** вносить любые изменения в Google Sheets таблицы (особенно лист Unit). Только чтение.
- Никогда не коммить `.env.local`
- Все API-ключи и токены хранятся **только** в `.env.local`
- Не удалять и не переименовывать важные файлы в `/src/lib/`
